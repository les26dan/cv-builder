# Local credentials (do not commit real values)

Store admin and third-party credentials in `.env.local` or a password manager.

Bootstrap admin (optional, for local dev):

- `BOOTSTRAP_ADMIN_EMAIL`
- `BOOTSTRAP_ADMIN_PASSWORD`
- `BOOTSTRAP_ADMIN_LOGIN_ALIAS` (optional, default `adminbuddy`)

See `docs/env-template.txt` and `SETUP_SUMMARY.md`.
