# CV Builder Landing Page – Project Plan

## 🎯 Objective
Build a lean, experiment-driven, cost-efficient landing page MVP that:
- Converts visitors into waitlist signups
- Educates users about ATS optimization and resume enhancement features
- Validates user interest via multivariate A/B testing

## 🧩 Key Deliverables
- ATS-optimized resume section
- Keyword matcher section
- One-click cover letter generator section
- Email capture / waitlist block
- A/B variants per block (title, description, CTA)

## 📊 Goals & Metrics
- Conversion Rate (CTR on CTA buttons)
- Email signup rate
- Time on page per variant
- Click-through rate per experiment variant

## 🎯 Success Metrics
- 25% clickthrough rate on primary CTA (based on variant)
- Clearly measure conversion rate (CTA click) of each variant
