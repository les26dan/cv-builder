Phụ lục này tổng hợp các test case kiểm thử chức năng (black-box testing) của hệ thống, bao gồm 24 test case bao phủ các module xác thực, tải lên CV, soạn thảo và xuất CV. Các test case được thực hiện thủ công theo kịch bản; với các component React quan trọng, khóa luận bổ sung unit test bằng Jest và Testing Library. Tất cả 24 test case đều đạt; chi tiết từng module được trình bày dưới đây.
